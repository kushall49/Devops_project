## 🎯 8 ENDPOINTS AT YOUR FINGERTIPS

| # | Endpoint | Method | Purpose | Parameters |
|---|----------|--------|---------|-----------|
| 1 | `/` | GET | Welcome info | None |
| 2 | `/demo-image` | GET | Get test image | None |
| 3 | `/health` | GET | Server status | None |
| 4 | `/functions` | GET | List functions | None |
| 5 | `/process/resize` | POST | Scale images | file, width, height, ratio, quality |
| 6 | `/process/enhance` | POST | Improve quality | file, brightness, contrast, sharpness, color |
| 7 | `/process/filter` | POST | Apply effects | file, filter |
| 8 | `/process/pipeline` | POST | All 3 steps | file, width, height, brightness, contrast, sharpness, color, filter |

## 💡 QUICK TIPS

**Tip 1: Start with /demo-image**
- No parameters needed
- Gives you a test image immediately
- Perfect starting point

**Tip 2: Use /process/pipeline for efficiency**
- Do resize + enhance + filter in ONE call
- Saves time vs multiple operations
- Better performance

**Tip 3: Reference the guides while exploring**
- Keep DOCUMENTATION_INDEX.md open
- Find specific info quickly
- Learn as you go

**Tip 4: Try extreme values to understand effects**
- brightness: 0.5 vs 2.0 (see difference)
- contrast: 0.5 vs 2.0 (see impact)
- Helps understand parameter ranges

**Tip 5: Use Presets**
- Instagram (brightness=1.15, contrast=1.25, color=1.3)
- Vintage (brightness=1.05, color=0.8, filter=sepia)
- B&W (contrast=1.4, filter=grayscale)
